## Roadmap

### Phase 1 — Base Setup
- Repository setup
- Vision + Roadmap added
- Folder structure creation
- MIT License setup

### Phase 2 — Browser Prototype
- Basic UI (tabs, address bar, home)
- Webview engine integration
- Navigation controls (back, forward, refresh)
- Secure HTTPS lock indicator

### Phase 3 — Web3 Integration
- Wallet connect integration
- .hind domain resolver system
- OTP-based blockchain identity
- Private browser mode (no tracking, zero cookies)

### Phase 4 — HindMail System
- Web3 mail ID (user@hind)
- OTP verification only (no selfie)
- Anti-spam engine (unverified sender blocked)
- Encrypted inbox system

### Phase 5 — HindStorage
- Encrypted personal cloud
- Bookmark & history sync
- Zero-trust encryption protocol

- Fake link detector

- Real-time threat scoring
